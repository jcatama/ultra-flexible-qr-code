== Description ==

Create, manage & share dynamic QR code content

If you need customization, please feel free to email me at jcatama.ph@gmail.com

== Features ==
* Manage dynamic QR content
* Downloadable QR image
* Copy redirect link
* Smart Redirect
** If the content is a URL, it will redirect to the given link
** If the content is not a URL, it will return a json response
